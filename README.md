# FitBuddy

FitBuddy is a cross-platform fitness tracking application built in C++ and Qt that enables users to log workouts, monitor progress, manage fitness goals, and visualize performance metrics through an intuitive and scalable interface. The application leverages object-oriented design principles and modular software architecture to provide a reliable and extensible fitness management solution.

---

##  Features

- Workout tracking and exercise logging  
- Personalized fitness goal management  
- Progress monitoring and performance analytics  
- User-friendly graphical interface built with Qt  
- Persistent data storage using SQLite  
- Modular and scalable system architecture  
- Cross-platform desktop support  
- Real-time fitness data updates  

---

## 🛠️ Technologies Used

- C++  
- Qt Framework  
- SQLite  
- Object-Oriented Programming (OOP)  
- Model-View-Controller (MVC) Design Principles  
- Event-Driven Programming  
- Git & GitHub  

---

##  Project Architecture

FitBuddy is designed using a modular architecture separating UI, business logic, and data layers for maintainability and scalability.

### Core Modules

- **User Management** – Handles profiles and user settings  
- **Workout Tracker** – Logs and manages workout sessions  
- **Goal System** – Tracks fitness objectives and milestones  
- **Analytics Engine** – Generates performance insights and trends  
- **Database Layer** – Manages persistent storage using SQLite  
- **UI Layer** – Qt-based interactive interface
